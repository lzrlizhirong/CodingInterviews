## 题目描述
一只青蛙一次可以跳上1级台阶，也可以跳上2级。求该青蛙跳上一个n级的台阶总共有多少种跳法。

## 解决方法
这一题本质上还是斐波那契数列

```java
public class Solution {
    public int JumpFloor(int target) {
        int a = 1; 
        int b = 1;
        while(--target>0){
            a = a + b;
            b = a - b;
        }
        return a;
    }
}

```
