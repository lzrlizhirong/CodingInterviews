## 题目描述
输入两个链表，找出它们的第一个公共结点。

## 解决方法

```java
/*
public class ListNode {
    int val;
    ListNode next = null;

    ListNode(int val) {
        this.val = val;
    }
}*/
public class Solution {
    public ListNode FindFirstCommonNode(ListNode pHead1, ListNode pHead2) {
		//先将两个头结点保存        
		ListNode p1 = pHead1;	
        ListNode p2 = pHead2;
		int length1 = 0;
        int length2 = 0;
		//计算两个链表的长度
        while(pHead1!=null){
            ++length1;
            pHead1=pHead1.next;
        } 
        while(pHead2!=null){
            ++length2;
            pHead2=pHead2.next;
        }
		//如果链表1的长度大于链表2的长度，先让链表1走（length1-length2）的长度，然后链表1和链表2一起走；反之，同理。
 		if(length1>length2){
            for(int i=0;i<length1-length2;i++){
                p1=p1.next;
            }
        }else{
            for(int i=0;i<length2-length1;i++){
                p2=p2.next;
        	}
   	 	}
        while(p1!=p2){
            p1=p1.next;
            p2=p2.next;
        }
        return p1;
    }
}
```
